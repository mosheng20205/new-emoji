# 火山空白窗口模板

优先使用 `examples/火山/src/new_emoji_接口.wsv` 中的 `_字节集` 方法。中文和 emoji 文本按 UTF-8 字节指针 + 长度传入。

## 界面内容

- 窗口标题：`✨ 火山空白窗口`
- 正文：`欢迎使用 new_emoji 🚀`
- 主按钮：`✅ 确认操作`

## 调用流程

```text
标题字节集 = UTF8("✨ 火山空白窗口")
正文字节集 = UTF8("欢迎使用 new_emoji 🚀")
按钮Emoji字节集 = UTF8("✅")
按钮文字字节集 = UTF8("确认操作")

窗口句柄 = NewEmoji接口类.创建窗口_字节集(标题字节指针, 标题字节长度, 240, 120, 860, 560, 0xFF2D7DFF)
根容器 = NewEmoji接口类.创建容器(窗口句柄, 0, 0, 0, 820, 500)
NewEmoji接口类.创建文本_字节集(窗口句柄, 根容器, 正文字节指针, 正文字节长度, 32, 32, 460, 40)
NewEmoji接口类.创建按钮_字节集(窗口句柄, 根容器, 按钮Emoji字节指针, 按钮Emoji字节长度, 按钮文字字节指针, 按钮文字字节长度, 32, 96, 160, 42)
NewEmoji接口类.显示窗口(窗口句柄, 1)
NewEmoji接口类.运行消息循环()
```

窗口和根容器尺寸已预留首屏余量，新增控件时要同步增大窗口或容器。
