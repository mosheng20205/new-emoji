# Aside 侧边栏

`Aside` 是 Container 套件中的侧边栏区域组件，当前状态：**已完成**。

| 项目 | 值 |
|---|---|
| 创建导出 | `EU_CreateAside` |
| 默认样式 | 背景 `#D3DCE6`，文字 `#333333`，文本居中 |
| 默认尺寸 | 宽度 200 |
| Python helper | `create_aside` |

相关 API：`EU_SetContainerRegionTextOptions`、`EU_GetContainerRegionTextOptions`、`EU_SetElementText`、`EU_SetPanelStyle`。

`Aside` 适合承载纵向 Menu、筛选器和导航树。放入横向 Container 时默认作为固定宽侧栏。
