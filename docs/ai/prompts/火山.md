# 火山 AI Prompt

```text
你正在使用 new_emoji.dll 为火山视窗生成 Windows 原生 UI。

请优先阅读：
1. examples/火山/src/new_emoji_接口.wsv
2. examples/火山/README.md
3. docs/ai/api_manifest.full.json
4. examples/templates/火山/空白窗口.md

规则：
- 优先使用 NewEmoji接口类 中已有的 _字节集 方法。
- 中文和 emoji 文本按 UTF-8 字节指针 + 长度传入。
- DLL 位数必须和火山程序位数一致。
- 界面文案必须中文化，窗口标题、主按钮、核心展示项必须有 emoji。
- 窗口和根容器首次打开时不能裁切控件，要保留至少 20px 逻辑余量。
```

窗口图标：使用 `EU_SetWindowIcon` 传入 UTF-8 `.ico` 路径字节，或使用 `EU_SetWindowIconFromBytes` 传入 `.ico` 文件完整字节。

窗口圆角：使用 `EU_SetWindowRoundedCorners(hwnd, enabled, radius)` 设置 Windows 窗口外形圆角；支持 DWM 的系统会优先使用平滑圆角，Win10 会回退为 per-pixel alpha 分层窗口并通过 `UpdateLayeredWindow(ULW_ALPHA)` 提交。
