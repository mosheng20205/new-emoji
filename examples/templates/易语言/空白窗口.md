# 易语言空白窗口模板

> 易语言 IDE 中不要直接写 emoji 或特殊 Unicode 字符，否则可能变成问号。下面的标题、正文、按钮 emoji、按钮文字都使用 UTF-8 字节集 + 长度。

## UTF-8 字节集

```text
标题字节集 ＝ { 226, 156, 168, 32, 230, 152, 147, 232, 175, 173, 232, 168, 128, 231, 169, 186, 231, 153, 189, 231, 170, 151, 229, 143, 163 }
标题长度 ＝ 25

正文字节集 ＝ { 230, 172, 162, 232, 191, 142, 228, 189, 191, 231, 148, 168, 32, 110, 101, 119, 95, 101, 109, 111, 106, 105, 32, 240, 159, 154, 128 }
正文长度 ＝ 27

按钮Emoji字节集 ＝ { 226, 156, 133 }
按钮Emoji长度 ＝ 3

按钮文字字节集 ＝ { 231, 161, 174, 232, 174, 164, 230, 147, 141, 228, 189, 156 }
按钮文字长度 ＝ 12
```

## 调用流程

```text
窗口句柄 ＝ 创建窗口 (标题字节集, 标题长度, 240, 120, 860, 560, 4297952767)
根容器 ＝ 创建容器 (窗口句柄, 0, 0, 0, 820, 500)
创建文本 (窗口句柄, 根容器, 正文字节集, 正文长度, 32, 32, 460, 40)
创建按钮 (窗口句柄, 根容器, 按钮Emoji字节集, 按钮Emoji长度, 按钮文字字节集, 按钮文字长度, 32, 96, 160, 42)
显示窗口 (窗口句柄, 1)
```

`.DLL命令` 左侧命令名使用中文，右侧 DLL 入口名保持 `EU_CreateWindow`、`EU_CreateContainer`、`EU_CreateText`、`EU_CreateButton`、`EU_ShowWindow`。
