# 火山 AI Prompt

```text
你正在使用 new_emoji.dll 为火山视窗生成 Windows 原生 UI。

请优先阅读：
1. examples/火山/new_emoji_module/src/new_emoji_接口.wsv
2. examples/火山/new_emoji_module/src/new_emoji_助手.wsv
3. examples/火山/README.md
4. docs/ai/api_manifest.full.json
5. examples/templates/火山/空白窗口.md

规则：
- 普通项目优先安装或引用 new_emoji.vcip 对应模块源；开发期可直接引用 examples/火山/new_emoji_module/new_emoji.vgrp。
- 优先使用 NewEmoji助手类 创建 91 个组件；需要底层能力时再调用 NewEmoji接口类。
- 中文和 emoji 文本按 UTF-8 字节指针 + 长度传入。
- DLL 位数必须和火山程序位数一致。
- 界面文案必须中文化，窗口标题、主按钮、核心展示项必须有 emoji。
- 窗口和根容器首次打开时不能裁切控件，要保留至少 20px 逻辑余量。
```

窗口图标：使用 `EU_SetWindowIcon` 传入 UTF-8 `.ico` 路径字节，或使用 `EU_SetWindowIconFromBytes` 传入 `.ico` 文件完整字节。

Table 普通文本单元格双击编辑使用 `EU_SetTableDoubleClickEdit` 开启；列/单元格覆盖使用 `EU_SetTableColumnDoubleClickEdit` / `EU_SetTableCellDoubleClickEdit`，编辑回调使用 `EU_SetTableCellEditCallback`。

窗口圆角：使用 `EU_SetWindowRoundedCorners(hwnd, enabled, radius)` 设置 Windows 窗口外形圆角；支持 DWM 的系统会优先使用平滑圆角，Win10 会回退为 per-pixel alpha 分层窗口并通过 `UpdateLayeredWindow(ULW_ALPHA)` 提交。
