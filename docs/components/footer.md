# Footer 底栏

`Footer` 是 Container 套件中的底栏区域组件，当前状态：**已完成**。

| 项目 | 值 |
|---|---|
| 创建导出 | `EU_CreateFooter` |
| 默认样式 | 背景 `#B3C0D1`，文字 `#333333`，文本居中 |
| 默认尺寸 | 高度 60 |
| Python helper | `create_footer` |

相关 API：`EU_SetContainerRegionTextOptions`、`EU_GetContainerRegionTextOptions`、`EU_SetElementText`、`EU_SetPanelStyle`。

`Footer` 适合承载状态栏、分页、底部操作区。放入启用流式布局的 Container 后，会参与自动纵向布局。
