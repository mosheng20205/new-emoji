# 火山空白窗口模板

优先安装 `new_emoji.vcip`，或在开发期引用 `examples/火山/new_emoji_module/new_emoji.vgrp`。普通界面代码优先使用 `NewEmoji助手类`，中文和 emoji 文本由助手类转换为 UTF-8 字节指针 + 长度。

## 界面内容

- 窗口标题：`✨ 火山空白窗口`
- 正文：`欢迎使用 new_emoji 🚀`
- 主按钮：`✅ 确认操作`

## 调用流程

```text
窗口句柄 = NewEmoji助手类.创建窗口("✨ 火山空白窗口", 240, 120, 860, 560, 0xFF2D7DFF)
根容器 = NewEmoji助手类.创建容器套件(窗口句柄, 0, 0, 0, 820, 500)
NewEmoji助手类.创建文本(窗口句柄, 根容器, "欢迎使用 new_emoji 🚀", 32, 32, 460, 40)
NewEmoji助手类.创建按钮(窗口句柄, 根容器, "✅", "确认操作", 32, 96, 160, 42)
NewEmoji助手类.显示窗口(窗口句柄, 1)
NewEmoji助手类.运行消息循环()
```

窗口和根容器尺寸已预留首屏余量，新增控件时要同步增大窗口或容器。
